## Terminology
- **MATN**= content of the "REWAYA"
- **SANAD**= people involved in the path until Prophet Mohamed SAWS
- **RAWI**= person having at least one **REWAYA**
- **REWAYA**= **MATN** + **SANAD**

## Main use cases
